What is HTML? HTML is the structure and content of a website. It is a declarative language

Proper syntax for HTML tags

the proper sintax of a tag looks like this

< h1 > I am a heading < / h1 >

The 'H1' in this element is the tag name

the 'H1' when placed within the < > is described as an open tag

in this case, ' I am a heading' is the element content

< / h1> is a closing tag. Closing tags must have a forward slash
