What is HTML?

Naming convention for all filenames, paths and folders

Proper naming conventions for all filenames, paths and folders are as follows.

  • There must be no spaces
  • must be all lowercase
  • and contain only letters, numbers and dashes (-)

These pictures were helpful resources to better understand and apply the naming conventions.

Best practices for commit messages

The best practices when commiting a message are:

  • Minimum 3 words & 10 characters
  • Proper spelling & grammar
  • Capital starting letter, no period
  • Must start with an imperative verb: Create, Fix, Add, Solve, etc.
  • Pretend every commit starts with the phrase: “This commit will…”

What is HTML?

HTML is the structure and content of a website. It is a declarative language

Proper Syntax for HTML tags

The proper sintax of a tag looks like this (below):

< h1 > I am a heading < / h1 >

The 'H1' in this element is the tag name

the 'H1' when placed within the < > is described as an open tag

In this case, ' I am a heading' is the element content

< / h1> is a closing tag. Closing tags must have a forward slash

Paragraph Tag

a paragraph tag < p > defines a paragraph of text. It is a block-level element and always starts on a new line.

Lists

There are three types of lists

  1. Unordered List:
    • These lists are by default put into bulltet form and and each element is listed using the li tag. For grouping a collection of items that do not have a numerical ordering, and their order in the list is meaningless
    • Uses the tag < ul >
  2. Ordered list:
    • The elements for this list are indicated using the li tag much like an unordered list. Unlike the unordered list however, the order is of importance and by default the first element of the list will begin with 1, chronilogically decending from there.
    • Uses the tag < ol >
  3. Descriptive List:
    • This list is seen in text like recipes and require a term (dt) and a description (dd) tag.
    • It's important to note that a dl can have more have one dd, but only one dt.

The a tag

This defines a hyperlink, which is used to link from one page to another. The most important part of an a element is the href atributs which tels the destination of the link.

the img tag

Images are not embedded in the website but are linked to the HTML document. Images need two attributes: the location of the image and the alternative content.

Figure (img & figcaption)

The HTML element < figure > is a self-contained content with an option caption, which is specified with the < figcaption >. The figure, its caption, and its contents are referenced as a single unit. The structre for that would look like this:
